Bulk Product Import

Import Instructions

Upload a CSV file with the following columns:

  • Name - Product name (required)
  • Barcode - Product barcode (required, numeric)
  • SellingPrice or Price - Selling price (required)
  • PurchasePrice or ActualPrice - Purchase/Cost price (optional)
  • GenericName - Generic/active name (optional)
  • Manufacturer - Manufacturer name (optional, will create new manufacturer if doesn't exist)
  • Supplier - Supplier (optional)
  • BatchNumber - Batch number (optional)
  • Category - Product category (optional, will use default if not specified)
  • Quantity - Initial stock quantity (optional, default: 0)
  • MinStock - Minimum stock level (optional, default: 1)
  • ExpirationDate - Expiry date (optional, formats supported: DD-MMM-YY, DD-MMM-YYYY, DD/MM/YYYY, YYYY-MM-DD)

Note: The first row should contain column headers. New categories and manufacturers will be created automatically if they don't exist.
